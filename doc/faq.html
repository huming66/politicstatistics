• Always use Google Chrome as the browser to open http://prodhpcts01/spa 
  • Locating a measurement
    • use the "Region > Area > Station > Elemnet" hierarchy at the left
    • use the Measurement hierarchy at the right
    • search any chracters in name of Region, Area, Station and Elemnet ... support "regular expression"
    • search any chracters in name of measures ... support "regular expression" ... e.g. .* .+ \d{3} [0-9]{3} \w{3} [A-z]{3} ... )
  • Select measure(s) for charting
    • select by single click (toggle) on (one or miltiple) the name of measure (NOT the [+] sign) will bring the data on chart
    • select the type of chart (scatter chart need two measures selected)
    • check the aggregate will remove all individue measure, and only keep the aggregate summation
    • single click (toggle) on the [+] sign will change the sign of the measure data
    • different aggregation calculation ... dropdown list and user-defined formula
  • Data export
    • only for time series data
    • to a local CSV file with timestamp
  • user-defined formula for aggregation calculation for miltiple measurements
    • support JavaScript built-in functions
    • use array v to represnt the selected measures ... e.g. v[0] the 1st measure, v[1] the 2nd measure ...
    • example: Math.max(...v) return the max measure in array v (for each point of time)
  • Data of other measures
    • As a starting point, this only includes hourly MW of all asset generation and flow of all lines
    • on-the-fly hourly data download (2013 to 2019 for now) for all listed PI-tags
    • Other data can be added as required ...
  • When data is not refreshing
    • Refresh browser; if still not working, then
    • Clear Chrome memory cache: [Ctrl + F5]; if still not working, then
    • Clear Chrome disk cache: Setting > Advanced > Privacy and security > Clear browsing data > Clear data
